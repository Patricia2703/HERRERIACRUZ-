
# 🛠️ Sistema de Gestión Empresarial - Herrería Cruz

Este es un sistema web integral diseñado para la administración, cotización y control de inventarios de **Herrería Cruz**. Permite gestionar clientes, capturar especificaciones técnicas de proyectos (puertas, zaguanes, ventanas), calcular costos de insumos de manera automática y exportar cotizaciones oficiales en formato PDF.

## 🚀 Características Principales
* **Módulo de Presupuestos:** Captura de medidas (alto, ancho, espesor) y generación de folios.
* **Cálculo de Insumos:** Desglosa dinámicamente los materiales asignados a cada trabajo.
* **Exportación Formal:** Generación de archivos PDF listos para entregar al cliente con los datos del negocio.
* **Panel de Administración (Estatus):** Flujo de aprobación y rechazo de órdenes conectadas al stock.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** Angular 17/18, Bootstrap 5, FontAwesome (iconos).
* **Backend:** Python (Flask), REST API.
* **Base de Datos:** MySQL (conectado vía ODBC).
* **Generación de Reportes:** ReportLab (Python).

## 💻 Instalación y Uso Local

### Frontend (Angular)
1. Entra a la carpeta del proyecto: `cd angular-project`
2. Instala las dependencias necesarias: `npm install`
3. Arranca el servidor local: `ng serve`
4. Abre en tu navegador: `http://localhost:4200`

### Backend (Python)
1. Asegúrate de tener instalado Flask y ReportLab: `pip install flask reportlab`
